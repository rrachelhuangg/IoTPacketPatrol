Selected Dataset for this Project
The data used for this project is a 733,705 element dataset curated by Dyuti_Dasmahaptra on Kaggle and can be found here. The dataset was selected for its quality and large size, containing network traffic flows collected from Internet of Things (IoT) devices that were operating in environments affected by botnet attacks. Each row in the dataset represents a single network flow with packet sequence ID, used protocol, source and destination IP addresses, source and destination ports, and a multitude of other characteristics of the flow. Each flow is labeled 0 if it a benign flow and 1 if it is a malicious attack.
The dataset was used to train a logistical regression model that classifies a flow as benign or malicious, and the specific categories and methods of the attack if applicable.
The dataset was stored in MongoDB's Atlas Database [...].
